# DN5-DeepSkilling - Advanced Track

Welcome to the deep-skilling track. This section focuses on advanced software engineering topics, enterprise architecture, and full-stack application development.

## Track Overview

This track builds on the foundations from `DN5-Upskilling` and is aimed at developers who want stronger backend, architecture, and frontend specialization skills.

- Duration: 12 to 16 weeks
- Level: Intermediate to Advanced
- Prerequisite: DN5-Upskilling or equivalent fundamentals

---

## Learning Outcomes

By completing this track, you will be able to:
- Implement SOLID principles and design patterns
- Analyze and optimize algorithms
- Write efficient SQL and tune queries
- Build data-driven applications with Entity Framework Core
- Create RESTful APIs with ASP.NET Core
- Design microservices-based systems
- Build interactive frontends with React
- Containerize and deploy applications
- Use Git and CI/CD pipelines effectively

---

## Modules

### 1. Design Patterns
- Duration: 1 to 2 weeks
- Focus: SOLID principles, GoF patterns, enterprise design
- Start here: [DesignPatterns/Notes.md](DesignPatterns/Notes.md)

### 2. Data Structures and Algorithms
- Duration: 2 to 3 weeks
- Focus: complexity, searching, sorting, trees, graphs
- Start here: [DSA/Notes.md](DSA/Notes.md)

### 3. Advanced SQL
- Duration: 1 to 2 weeks
- Focus: indexing, execution plans, advanced queries, performance tuning
- Start here: [AdvancedSQL/Notes.md](AdvancedSQL/Notes.md)

### 4. Entity Framework Core
- Duration: 2 weeks
- Focus: LINQ, migrations, relationships, data access patterns
- Start here: [EntityFrameworkCore/Notes.md](EntityFrameworkCore/Notes.md)

### 5. ASP.NET Core Web API
- Duration: 2 to 3 weeks
- Focus: API design, middleware, validation, authentication
- Start here: [ASPNetCoreWebAPI/Notes.md](ASPNetCoreWebAPI/Notes.md)

### 6. Microservices
- Duration: 2 weeks
- Focus: service boundaries, resilience, communication, orchestration
- Start here: [Microservices/Notes.md](Microservices/Notes.md)

### 7. React
- Duration: 2 to 3 weeks
- Focus: components, hooks, routing, forms, API integration, state management
- Start here: [React/Notes.md](React/Notes.md)

### 8. Docker
- Duration: 1 to 2 weeks
- Focus: containerization, Dockerfiles, Compose, deployment basics

### 9. Git and CI/CD
- Duration: 1 to 2 weeks
- Focus: branching workflows, automation, pipelines, deployment practices
- Start here: [Git-CICD/Notes.md](Git-CICD/Notes.md)

1. Design Patterns -> DSA
