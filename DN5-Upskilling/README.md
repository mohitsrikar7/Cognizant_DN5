# DN5-Upskilling - Foundation Track

Welcome to the DN5-Upskilling track! This is your foundation for becoming a professional web and software developer.

## 📋 Track Overview

The Upskilling track covers the essential technologies needed to build modern web applications and understand core programming principles. This track is designed for beginners and focuses on building strong fundamentals.

**Duration:** 8-12 weeks (self-paced)
**Level:** Beginner to Intermediate
**Prerequisites:** None (basic computer literacy assumed)

---

## 🎯 Learning Outcomes

By completing this track, you will be able to:

✅ Create semantic and accessible HTML5 documents
✅ Style web pages with CSS3 using modern layout techniques
✅ Write interactive JavaScript applications using ES6+
✅ Design and query relational databases with SQL
✅ Understand Object-Oriented Programming principles in C#
✅ Build small projects combining all these technologies

## 📊 Recommended Learning Path

**Phase 1: Web Fundamentals (4-5 weeks)**
1. HTML5 → CSS3 → JavaScript basics
2. Build: Simple interactive website

1. SQL fundamentals
2. C# language and OOP

**Phase 3: Integration Project (2-3 weeks)**
1. Build a complete application using all technologies

✅ Type all code manually (no copy-paste)
